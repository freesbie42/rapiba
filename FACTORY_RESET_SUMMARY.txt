🔧 ALTERNATIVES FOR SELECTIVE CLEANUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Only clear logs (keep backups & config):
    sudo rm -rf /var/log/rapiba/*
    sudo systemctl restart rapiba

  Only reset duplicate database:
    sudo rm /var/lib/rapiba/duplicate_db.sqlite3
    sudo systemctl restart rapiba

  Only delete backups (keep logs & config):
    sudo rm -rf /backup/*

  Only reset configuration (keep everything else):
    sudo rm /etc/rapiba/rapiba.conf
    sudo cp rapiba/etc/rapiba.conf /etc/rapiba/
    sudo systemctl restart rapiba
